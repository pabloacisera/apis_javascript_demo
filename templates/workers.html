Workers

web workes

Es un script de JavaScript que se ejecuta en un hilo separado del hilo principal del navegador. Su propósito es realizar tareas pesadas sin bloquear la interfaz de usuario. Se ejecuta en segundo plano y no tiene acceso al dom solo se comunica con él hilo principal, no sobrevive al cierre de la pestaña o ventana


server worker

Es un tipo especial de worker que actúa como un proxy entre el navegador y la red. Su propósito principal es permitir aplicaciones web progresivas (PWA) y funcionalidad offline. Permite cache y notificaciones push, y persiste aún si cierras las pestañas
